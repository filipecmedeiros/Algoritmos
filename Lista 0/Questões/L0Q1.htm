L0Q1 - Soma

  • Entrada: L0Q1.in
  • Saída: L0Q1.out
  • Clarifications
Some A com B

Formato de Entrada

A primeira linha do input, será um inteiro n que indicará a quantidade de casos de teste (1 <= n <= 1000), n linhas seguirão com dois inteiros positivos A e B. E A e B terão até 10^5 dígitos

Formato de Saída

Para cada caso de teste imprima o resultado da soma.

Exemplos

Entrada (download):

3
1 1
3 3
1073741823 1073741823
Saída (download):

2
6
2147483646
