L0Q2 - Integralandia x Nlogonia

  • Entrada: L0Q2.in
  • Saída: L0Q2.out
  • Clarifications

	Nlogonia está em guerra com Integralândia. Em tempos de guerra, qualquer informação que caia em mãos erradas pode significar a perda de uma batalha. Nlogonia tem uma tecnologia muito interessante, que de certa forma, encripta o horário dos seus ataques (Integralândia não sabe desse tipo de tecnlogia, mas não pense que ela é mais fraca por causa disso), esse aparelho é o Relógio Binário.





	O Relógio Binário, é um tipo de relógio que mostra a hora num formato binário, a partir de um formato sexagisemal (apenas 6 bits importam). Os relógios binários usam 3 colunas de LED para representar zeros(0) e uns (1), e cada coluna (ou linha) representa uma unidade de tempo (hora, minuto ou segundo).





	Esses relógios são famososo e Nlogonia e cada soldado quando vai pra guerra ganha um relógio novinho. Os soldados gostam de usar esse relógio de duas maneiras:




    
	
  1. Colunas de LED, dispostas verticalmente.
    2. 	
	
  2. Colunas de LED, dispostas horizontalmente.
    3.

Formato de Entrada

A primeira linha do input será um único inteiro N (1 <= N <= 1000) que dirá o número de casos de testes. Cada caso de teste conterá o horário no formato sexagesimal.

Formato de Saída

Para cada caso de teste você deverá imprimir APENAS uma linha com os seguintes valores: O número do caso de teste (valor decimal, e começando a partir de 1), um espaço, o valor binário no formato vertical (18 valores binários), um espaço, o valor binário no formato horizontal (também 18 valores binários).

Exemplos

Entrada (download):

2
10:37:49
00:00:01
Saída (download):

1 011001100010100011 001010100101110001
2 000000000000000001 000000000000000001
