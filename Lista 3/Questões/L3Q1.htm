L3Q1 - Hugo, o observador.

  • Entrada: L3Q1.in
  • Saída: L3Q1.out
  • Clarifications
	Uma das conclusões mais importantes da ciência do século XX é que o nosso universo tem, atrás de si, uma longa história. Tal como os humanos observadores de estrelas, também as estrelas nascem, vivem e, finalmente, morrem.
	O que permite o encontro de nós humanos e as estrelas,apesar de tão distante delas, é a luz. Quando observamos o céu, vemos apenas uns pontos brilhantes, porém com o advento do telescópio revelou-se muito o número de estrelas.  
	Hugo Freire, um observador de estrelas, toda a noite na varanda do seu apartamento (Hugo mora beira-mar) ele tem o seguinte hábito: Se duas estrelas estão perto o suficiente uma da outra, elas devem pertencer a mesma constelação. Hugo quer nomear essas constelações mas como ele é amador, ele tem medo de botar nomes aleatórios. Então, você pode ajuda-lo pelo menos contar quantas constelações há no céu?
	
	Observação: Duas estrelas pertencem a mesma constelação se a distância entre as suas projeções em um plano bidimensional não é mais que X unidades de comprimento.

Formato de Entrada

Na primeira linha será um inteiro T(T ≤ 50) que é o número de casos de testes. Nas T linhas seguintes cada caso conterá o número de estrelas N(N≤ 1000 ) e a distancia D(0.00 <= D <= 1000.00), em seguida nas N linhas seguintes se passa cada ponto (x,y) da  estrela (−1000.00 ≤ X, Y ≤ 1000.00).

Formato de Saída

Para cada caso teste,  é impresso uma linha :”Caso K: NUM”, onde  K (iniciando de 1) é o número do caso e NUM o número de constelações. Olhem o caso exemplo para mais informações.
